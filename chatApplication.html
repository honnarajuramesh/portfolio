Chat Application

The purpose of this chat application is to allow users be able to the chat with each other,and in group.

Use

The functionality of the chat application is to give the user ability to chat with whoever online on the application. Group chat is also added.

Login page

The Login page will handled by conditional rendring, Once the user Logs in the data will be stored in the Local storage and used for auto login.

We are using Rest API (chatengine.io) for storing the data and used in login.
